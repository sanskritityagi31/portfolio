Time Travelers Archive

  • Tech Stack: React, Node.js, MongoDB, OpenAI API, AWS EC2, GitHub Actions
  • GitHub URL: Project Link

Built a semantic document search platform enabling users to explore historical archives using both keyword-based and vector similarity search.

Indexed over 5,000 documents using OpenAI embeddings, optimized query paths to reduce latency, and deployed the system on AWS EC2 with automated CI/CD via GitHub Actions.
